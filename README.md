# Mental_Wellness_Chatbot
Serenity is an AI-powered mental health and wellness web app that offers an empathetic chat companion using Google Gemini AI, mood tracking with visual insights, guided mindfulness exercises, and access to crisis support resources. It focuses on accessible emotional support and well-being.
